انواع بارکد

  • Code 128: پشتیبانی کامل از کاراکترها، مناسب برای صنعتی
  • Code 39: حروف و اعداد، استفاده صنعتی گسترده
  • EAN-13: 13 رقم، استاندارد فروشگاهی جهانی
  • EAN-8: 8 رقم، برای محصولات کوچک
  • UPC-A: 12 رقم، استاندارد آمریکای شمالی
  • QR Code: داده‌های بزرگ، URL، و متن فارسی

میانبرهای صفحه کلید

  • Ctrl + Enter - تولید بارکد
  • Ctrl + S - ذخیره بارکد
  • Ctrl + P - چاپ بارکد
  • Esc - بستن پنجره‌های باز

نکات مفید

  • برای QR Code می‌توانید متن فارسی و URL وارد کنید
  • بارکدهای EAN و UPC فقط اعداد را قبول می‌کنند
  • می‌توانید چندین بارکد را به صورت گروهی تولید کنید
  • از فایل CSV برای وارد کردن داده‌های زیاد استفاده کنید
